# 医学知识库数据说明文档

## 数据来源

**原始项目**: QASystemOnMedicalKG  
**项目地址**: https://github.com/liuhuanyong/QASystemOnMedicalKG  
**作者**: 刘焕勇 (liuhuanyong)  
**数据文件**: `data/medical.json`  
**许可证**: 开源项目，仅供学习和研究使用

### 项目简介

QASystemOnMedicalKG 是一个基于医疗知识图谱的自动问答系统项目。该项目从无到有构建了一个以疾病为中心的医疗领域知识图谱，包含：

- **实体规模**: 约 4.4 万个知识实体
- **关系规模**: 约 30 万个实体关系
- **实体类型**: 7 类（疾病、症状、药品、食物、检查、科室、药品生产商）
- **关系类型**: 11 类（属于、常用药品、宜吃、忌吃、推荐食谱、所需检查、疾病症状、并发疾病等）

### 数据采集方式

根据原始项目说明，数据来源于：
- **垂直型医药网站**: 通过爬虫采集公开的医疗健康信息
- **数据格式**: 网页结构化数据（通过 XPath 解析提取）
- **数据范围**: 疾病百科、症状说明、治疗方案、用药指导等

## 数据结构说明

每条疾病记录包含以下字段：

### 核心信息字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `name` | string | 疾病名称 | "肺泡蛋白质沉积症" |
| `desc` | string | 疾病简介/详细描述 | 疾病的定义、特征、发病人群等 |
| `category` | array | 疾病分类 | ["疾病百科", "内科", "呼吸内科"] |

### 症状与病因字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `symptom` | array | 症状列表 | ["紫绀", "胸痛", "呼吸困难"] |
| `cause` | string | 病因说明 | 详细的致病原因和发病机制 |

### 治疗相关字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `cure_department` | array | 推荐就诊科室 | ["内科", "呼吸内科"] |
| `cure_way` | array | 治疗方式 | ["药物治疗", "支持性治疗"] |
| `cure_lasttime` | string | 治疗周期 | "约3个月" |
| `cured_prob` | string | 治愈概率 | "约40%" |

### 检查与用药字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `check` | array | 相关检查项目 | ["胸部CT检查", "肺活检"] |
| `common_drug` | array | 常用药物 | ["布地奈德气雾剂"] |
| `recommand_drug` | array | 推荐药物 | ["信谊布地奈德气雾剂"] |
| `drug_detail` | array | 详细药品信息 | 包含厂家和商品名的药品列表 |

### 饮食建议字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `do_eat` | array | 宜吃食物 | ["鸡蛋", "海蚌", "圆白菜"] |
| `not_eat` | array | 忌吃食物 | ["海蟹", "海虾", "海螺"] |
| `recommand_eat` | array | 推荐食谱 | ["紫苏粥", "菊花芦根茶"] |

### 预防与流行病学字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `prevent` | string | 预防措施 | 详细的预防建议 |
| `get_prob` | string | 发病率 | "0.00002%" |
| `get_way` | string | 传播途径 | "无传染性" |
| `easy_get` | string | 易感人群 | "多见于小儿" |
| `yibao_status` | string | 医保状态 | "是"/"否" |

### 并发症字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `acompany` | array | 常见并发症 | ["多重肺部感染", "支气管哮喘"] |

### 费用字段

| 字段名 | 类型 | 说明 | 示例 |
|--------|------|------|------|
| `cost_money` | string | 治疗费用参考 | "市三甲医院约（8000-15000元）" |

## 数据使用说明

### 适用场景

1. **医疗导诊系统** - 基于症状推荐就诊科室
2. **RAG知识检索** - 为LLM提供医学知识上下文
3. **疾病问答系统** - 回答用户关于疾病的咨询
4. **健康科普** - 生成疾病相关的科普内容
5. **医学研究** - 作为医疗NLP研究的基准数据

### 使用限制

⚠️ **重要声明**:

1. **非医疗建议**: 本数据仅供技术开发和研究使用，**不构成医疗建议**
2. **数据时效性**: 数据来源于2018年左右，部分医学信息可能已更新
3. **准确性限制**: 数据来源于公开网络，可能存在不准确或不完善之处
4. **专业指导**: 实际就医请遵循专业医生的指导

### 合规使用

根据原始项目作者的声明：
- 数据请勿用于商业用途
- 如侵犯相关单位权益，请联系原作者删除
- 本数据仅供学习和研究使用

## 技术处理建议

### 数据加载

```python
import json

# JSON Lines 格式读取
diseases = []
with open('knowledge_base/medical.json', 'r', encoding='utf-8') as f:
    for line in f:
        if line.strip():
            diseases.append(json.loads(line))
```

### RAG向量化

建议对以下字段进行向量化处理：
- `name` - 疾病名称
- `desc` - 疾病描述
- `symptom` - 症状列表
- `cause` - 病因

### 数据清洗建议

1. 处理缺失字段（部分记录可能缺少某些字段）
2. 标准化科室名称
3. 清洗症状描述中的特殊字符
4. 处理重复或相似的疾病条目
