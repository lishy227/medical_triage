### 2. 消息发送流程

```

### 3. 阶段更新流程

         │
         ▼

┌─────────────────┐
│   (后续阶段)    │────► 未开始阶段
│                 │      显示数字
└─────────────────┘

## 四、API接口

| 接口 | 方法 | 参数 | 返回值 | 说明 |
|------|------|------|--------|------|
| `/api/welcome` | GET | `session_id` | `{message, session_id}` | 获取欢迎消息 |
| `/api/chat` | POST | `{session_id, message}` | `{message, is_complete, stage, records}` | 发送消息 |
| `/api/reset` | POST | `{session_id}` | `{message, session_id}` | 重置对话 |

## 六、文件结构

```
web/
├── index.html          # 页面结构
│   ├── sidebar         # 侧边栏(阶段指示器)
│   ├── chat-container  # 聊天区域
│   └── loading-overlay # 加载动画
│

│   ├── CSS变量定义     # 颜色、阴影等
│   ├── 布局样式        # Flex布局

│   └── 动画效果        # 淡入、打字指示器等
│

    ├── 状态管理        # sessionId, isWaiting等
    ├── API调用         # fetch请求封装

    ├── 事件处理        # 点击、键盘事件
